# Nest.js SWC demo

This project is a basic Nest.js project which has support added to be compiled via 
[swc](https://github.com/swc-project/swc).

## How to use

```sh
# Build via default "nest" command
npm run build

# Build by using "swc"
npm run build:swc
```

```sh
# Run build application
npm run build:prod
```
